**IT Landscape Framework**

-A modern, interactive, and responsive visualization of the IT industry landscape. This single-page application navigates through various sectors of IT (Security, DevOps, Cloud, Development, etc.) and provides direct links to industry-standard tools.

-Live Demo https://martino-95.github.io/IT-tool-map/


**Features**

-Interactive Tree View: Expandable/collapsible categories.

-Search: Real-time filtering of tools and categories.

-Responsive Design: Works seamlessly on desktop and mobile.

-Modern UI: Dark mode aesthetic with glassmorphism effects.

-Zero Dependencies: Runs entirely in the browser using ES Modules (React + Tailwind).


**Installation & Usage**

-Option 1: Run Locally

--Download the index.html file.

--Open it in any modern web browser (Chrome, Firefox, Edge, Safari).

--That's it! No npm install or build servers required.



**Option 2:** Host on GitHub Pages (Free)

-Fork or clone this repository.

-Go to your repository Settings.

-Navigate to Pages (in the left sidebar).

-Under Build and deployment -> Branch, select main (or master) and hit Save.

-GitHub will provide you with a live URL (e.g., https://your-username.github.io/repo-name).


**Tech Stack**

-HTML5

-Tailwind CSS (via CDN)

-React 18 (via ESM.sh CDN)

-Lucide React (Icons)
